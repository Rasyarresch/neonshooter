# Neon Shooter

A fast-paced 2D arcade space shooter built with **HTML5 Canvas** and **Vanilla JavaScript**.

 **Play the prototype:** https://neonshooter-delta.vercel.app/

---

## 📖 About The Game

**Neon Shooter** puts you in control of a glowing spacecraft fighting through increasingly dangerous waves of enemies. Dodge incoming attacks, survive as long as possible, and rack up the highest score while navigating a vibrant neon-inspired battlefield.

Designed as a lightweight browser game, Neon Shooter focuses on responsive controls, satisfying combat, and progressive difficulty without relying on external game engines.

---

## Features

### Progressive Wave System

Face increasingly challenging enemy waves powered by a custom level progression system.

### Fast-Paced Arcade Combat

Destroy enemies, avoid incoming projectiles, and survive against overwhelming odds.

### Score-Based Gameplay

Compete against yourself and others by achieving the highest score possible.

## 🛠️ Tech Stack

| Technology                | Purpose                                   |

| HTML5 Canvas              | Rendering graphics and visual effects     |

| CSS3                      | UI styling and layout                     |

## 📂 Project Structure

```text
NeonShooter/
│
├── index.html          # Main entry point and game canvas
├── style.css           # UI and visual styling
├── LevelSystem.js      # Enemy wave progression and difficulty scaling
├── game.js             # Main game loop and state management
├── player.js           # Player movement and shooting logic
├── enemy.js            # Enemy behavior and spawning
└── assets/             # Images, sounds, and game resources
```
